### Key Features

#### 1. Offline Catalog
- Products cached in Isar
- Images cached locally
- Search works offline
- Auto-sync when online

#### 2. Telegram OTP
- 4-digit code sent via Telegram bot
- 5 minute expiry
- 3 attempts max
- Visual countdown timer

#### 3. Quick Order
- Select customer
- Add products from catalog
- Auto-calculate total
- Submit order (syncs when online)

## Usage

### Navigate to Catalog
```dart
Navigator.push(
  context,
  MaterialPageRoute(builder: (_) => const CatalogScreen()),
);
```

### Register Customer
```dart
Navigator.push(
  context,
  MaterialPageRoute(
    builder: (_) => const CustomerRegistrationScreen(
      salesRepId: 'sales_rep_001',
    ),
  ),
);
```

```dart
  context,
```
